# 短视频解析工具

## 简介
本工具用于解析短视频平台的视频链接，获取视频的详细信息，如视频地址、封面图、作者信息等。

## 功能特点
- 支持多种短视频平台的链接解析
- 快速获取视频相关信息
- 返回结构化的 JSON 数据

## 使用方法

### 请求参数
| 参数名 | 类型 | 描述 | 是否必填 |
| ---- | ---- | ---- | ---- |
| url | 字符串 | 短视频平台的视频链接 | 是 |

### 请求示例
```plaintext
[https://api.bugpk.com/api/xhs?url=http://xhslink.com/a/63LnyN3WImLb](https://api.bugpk.com/api/xhs?url=http://xhslink.com/a/63LnyN3WImLbb)
